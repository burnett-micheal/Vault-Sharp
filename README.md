# Vault Sharp
**An open-source, local-first platform that bridges the gap between the Data and UI layer.**

Built with C# + SQLite + Obsidian. 

## Tech Stack
- **C# Backend** - Command routing, logic, automation
- **SQLite Database** - Local data storage
- **Obsidian** - Markdown-based UI
- **VSCode** - JSON template editing
- **URI Protocol** - `vaultbridge://` commands
